Setting Up A Custom Domain

Part I: No One Enjoys Setting Up DNS

When you first get started with Publishist, your publication's new website will have a domain like "example.publishist.com". If you own your own domain (for example, let's say you own "www.examplemag.com") and you want to point it at your Publishist website, you'll need to configure DNS.

To set up DNS, it helps to know a little bit about how it works. If you already know a good bit, feel free to skip to the next section.
